🚀 Démarrage Essentiel

easy-market est une marketplace digitale connectée à une API REST. Elle permet aux vendeurs de publier des produits et aux clients de les acheter.

1

Ouvrir l'application

Lancez navigate.html dans votre navigateur. L'écran d'accueil affiche un marché animé en arrière-plan (canvas) avec la navigation principale en haut.

2

Barre de recherche globale

En haut, tapez un produit et appuyez sur Entrée pour chercher dans le catalogue. La recherche vous amène directement à la section Catalogue.

3

Navigation par onglets

Six sections disponibles : Accueil, Compte, Notifications, 📍 Localisation, Catalogue, Panier.

4

Plein écran

Cliquez sur le bouton ⛶ en haut à droite pour basculer en plein écran.

👤 Créer un compte Obligatoire

1

Authentification

Renseignez votre téléphone, nom et mot de passe (min. 6 caractères). Cliquez sur Créer un compte ou Connexion si vous avez déjà un compte.

Le téléphone sert d'identifiant unique. Exemple : 770000000

2

Choisir un rôle

Sélectionnez votre rôle :

Client — Achetez des produits sur le marketplace

Vendeur — Publiez et vendez vos produits

Cliquez sur Valider pour confirmer.

3

Token d'authentification

Après connexion, un token JWT est généré automatiquement. Il est utilisé pour toutes les requêtes API authentifiées. Vous pouvez le voir dans le champ Token en bas de la section Compte.

🏪 Devenir vendeur Vendeur

📋 Profil vendeur

1

Compléter votre profil

Renseignez : Nom complet, National ID, Ville, Activité (ex: chaussures, électronique) et WhatsApp (pour être contacté par les clients).

Cliquez sur Enregistrer pour sauvegarder.

📦 Publier un produit

2

Créer un produit

Remplissez : Nom (ex: iPhone 15), Prix ($), Description, URL image (optionnelle). Cliquez sur Publier.

📝 Publier un article

3

Créer un article

Les articles sont des publications qui permettent de démarrer une conversation avec les clients.

Remplissez Titre et Contenu, puis cliquez sur Publier.

💳 Abonnement vendeur

4

Période d'essai et abonnement

🆕 Les nouveaux vendeurs peuvent vendre pendant 14 jours sans abonnement.

Après l'essai, un abonnement est requis pour continuer à vendre :

  1. Sélectionnez la méthode de paiement (Simulé, Orange Money, M-Pesa, Airtel Money)
  2. Cliquez sur Payer pour créer l'abonnement
  3. Une référence est générée
  4. Cliquez sur Webhook pour simuler le paiement

🛒 Côté client Client

✉️ Contacter un vendeur

1

Messagerie article

Après qu'un vendeur a publié un article, vous pouvez envoyer un message :

  1. Récupérez l'ID de l'article (généré après publication par le vendeur)
  2. Écrivez votre message
  3. Cliquez sur Envoyer

Utilisez Inbox et Outbox pour voir vos messages.

🛍️ Catalogue et recherche Client & Vendeur

1

Rechercher des produits

Utilisez la barre de recherche en haut ou les filtres dans la section Catalogue :

  • Recherche : mots-clés (ex: iphone, chaussures)
  • Vendeur ID : filtrer par vendeur spécifique
  • Prix min / max : filtrer par fourchette de prix
2

Trier les résultats

Utilisez les boutons de tri au-dessus des produits :

Tous · 🔥 Populaires · ⭐ Meilleures ventes · 💰 Prix ↑ · 💰 Prix ↓

3

Ajouter au panier

Cliquez sur Ajouter sur une carte produit pour le sélectionner dans le panier. Le produit ID est automatiquement pré-rempli.

4

Remplir le catalogue

Cliquez sur Remplir pour générer des produits de démonstration via l'API.

🛒 Panier et commande Client

1

Ajouter un produit

Dans la section Panier, le produit ID est pré-rempli si vous avez cliqué sur Ajouter depuis le catalogue. Vous pouvez aussi le saisir manuellement.

  • Date : automatiquement remplie
  • Produit ID : l'identifiant du produit
  • Quantité : nombre d'articles
2

Checkout

Cliquez sur le bouton 🛒 Checkout pour passer la commande. Les articles du panier sont envoyés à l'API.

3

Voir le panier

Cliquez sur Voir pour consulter les articles actuellement dans votre panier.

📍 Localisation et carte Client & Vendeur

1

Me localiser

Cliquez sur 📍 Me localiser pour obtenir votre position GPS. Les coordonnées (latitude, longitude) sont affichées et enregistrées sur le serveur.

Vous pouvez aussi utiliser le bouton 📍 en bas à droite de l'écran (FAB).

2

Carte OpenStreetMap

La carte affiche votre position avec un marqueur violet. Cliquez sur 🗺️ Ouvrir pour centrer la carte sur votre position.

3

Vendeurs à proximité

Cherchez les vendeurs autour de vous :

  1. Activez votre position (étape 1)
  2. Définissez un rayon en kilomètres (défaut: 50 km)
  3. Cliquez sur 🔍 Rechercher
  4. Les vendeurs apparaissent sur la carte avec des marqueurs jaunes
  5. Cliquez sur un vendeur pour voir son itinéraire 🚗

🔔 Notifications Client & Vendeur

1

Activer les notifications

Cliquez sur 🔔 Activer pour demander la permission d'envoyer des notifications push.

2

Tester une notification

Remplissez le Type (Message, Commande, Paiement, Système), la Priorité, le Titre et le Contenu, puis cliquez sur 📨 Tester.

⚙️ Informations techniques Développeur

Architecture

  • Frontend : HTML + CSS + JavaScript vanilla, canvas animé en arrière-plan
  • Backend : API REST sur localhost:8000
  • Authentification : Firebase Auth + tokens JWT
  • Base de données : gérée par l'API backend
  • Carte : OpenStreetMap avec Leaflet.js
  • PWA : Installation sur mobile via manifest.json + service worker

Points de terminaison API utilisés

  • POST /auth/firebase — Échange token Firebase → JWT
  • POST /auth/register-vendor — Inscription vendeur
  • POST /role/select — Sélection du rôle
  • POST /vendors/register-profile — Profil vendeur
  • POST /subscriptions/create — Création abonnement
  • POST /subscriptions/webhook — Simulation paiement
  • GET /products/search — Recherche produits
  • POST /products/create — Création produit
  • POST /products/seed — Remplir catalogue
  • POST /articles — Création article
  • GET /articles/{id}/messages — Messages
  • POST /cart/add — Ajouter au panier
  • GET /cart — Voir le panier
  • POST /orders/checkout — Commander
  • POST /location/update — Mettre à jour position
  • GET /vendors/nearby — Vendeurs à proximité
